Seemless integration of "Syntax Highlighter and Code Colorizer for Wordpress" into TinyMCE

== Description ==

SyntaxHL-Editor offers an easy to use graphical user interface for 
[Syntax Highlighter and Code Colorizer for Wordpress](http://www.lastengine.com/syntax-highlighter-wordpress-plugin/). 
You can easily enter and modify source-code highlighted in your posts as well as specify options
for every single snippet.

It's based on [RichGuk's SyntaxHL plugin for TinyMCE](http://27smiles.com/projects/) and 
the [TinyMCE Valid Elements Wordpress plugin](http://www.engfers.com/plugins/tinymce-valid-elements/). 

== Installation ==

Make sure you have "Syntax Highlighter and Code Colorizer for Wordpress" installed
and activated. Then upload the plugin to your blog an activate it.
